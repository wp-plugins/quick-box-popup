== Description ==

A quicklook into Quick Box Popup

    ★ Create a light weight quick onclick popup box 
    ★ Full control on popup content
    ★ Standard WordPress content editor
    ★ Display based on browsed number of pages
    ★ Css settings like color, z-index, border etc
    ★ Shortcode for displaying popup in specific pages
    ★ Display specificaly in pages/posts/home page

This plugin allows you to create a quick box popup window with custom content in your site. You can customize the popup display by configuring various settings such as display logic settings and style settings(z-index, color, border etc). The plugin supports automatic and manual (shortcode) display. You can use the plugin to display any type of contents such as special promotions, subscription forms, social media icons, feedback forms, video presentations and much more.

= 1. What is a "quick box popup" ? =

A quick box popup is used to display any content inside a popup within a webpage. It is normally used to show subscription forms, contact forms, promotions, etc.

= 2. The Quick Box Popup plugin is not working properly =

Please check the wordpress version you are using. Make sure it meets the minimum version recommended by us. Make sure all files of the `Quick Box Popup` plugin are uploaded to the folder `wp-content/plugins/`

= 3. Can I use this plugin with other popup plugins ? =

You can use it with any other popup related plugins from XYZScripts. However it may interfere with popup plugins from other vendors.

= 4. Can I set repeat interval in minutes ? =

Yes, you can set repeat interval in hours as well as minutes.

= 5. What kind of contents can be placed in a popup? =

You may  use the popup to display any type of contents such as special promotions, subscription forms, feedback forms, video presentations and much more.

= 6. Can I display popup in specific pages only ? =

Yes, you can do this by using the shortcode provided from settings page.

= 7. Does it work with caching plugins ? =

You can turn on compatibility with cache plugins from settings page. Please  note that since this method uses ajax, the popup will be a rendered a bit slower than in normal mode.

= 8. Can I create multiple popup boxes for different pages ? =

You need to purchase the premium version for having this feature.
